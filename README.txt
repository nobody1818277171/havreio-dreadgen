This is a program for automatically creating dreadnought type tanks in havre.io by taking in weapon and body parts.

Requirements:
    -Node.js (no other dependecies)
    -Functioning brain

How to use:
    -Create a weapon or a body
    -The program uses the upgradeMessage field to let the user store information necessary for the script to function:

       
                        w-Pacifier-2
                        |    |     |
        part type (w or b)   |     |
weapon or body it upgrades from    |
                                tier

        · You can also do:

                        w-Part1,Part2,Part3,Part4...-2,

        which makes it an option for those parts

----------IMPORTANT------------------------------------------

        · When creating a tier 1 part, make it upgrade from *Root*:

                        w-Root-1
        This would be the upgradeMessage all tier 1 weapons use.
        This makes it upgrade from the Dreadnoughts root tank which upgrades from all your tanks in your preset.

---------------------------------------------------------------

    -You can then download the part as a tank and place them in the weps or body folders depending on what they are.

    -Start the script by opening this folder in your terminal and run "node dreadGen.js". after it runs
    the autogenerated.json file inside of out/ will be created or overwritten with a build of the preset.

Notes:
    -you may change config.json to configure start level, level per tier and how many tiers there are. (WARNING, THE CODE THAT USES THIS IS CURRENTLY HARDCODED DUE TO SOME ISSUES, TO EXTEND THE AMOUNT OF TIERS ADD MORE WBarrays to map
    -you can change the base files inside of the resource folder, but keep the file names the same.
